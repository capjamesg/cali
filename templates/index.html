Welcome to the Cali Home Page

Cali is a Discord bot run by jamesg.blog.

Cali connects with services like IndieWeb Search and jamesg.blog's personal blog infrastructure to send reminders and notifications via Discord.

Here are a few examples of what Cali can do:

  • Send a URL for an IndieWeb Search query.
  • Respond to a few keyword phrases like "make a sandwich" and "yay".
  • Post to a personal website that supports Micropub (as long as you provide an authentication token when setting up the bot).
  • Keep track of karma
  • And more...

Can I use Cali?

Cali is not currently used in any public channels so you cannot interact directly with Cali.

You can find the code powering Cali on the project GitHub repository.

To add a new command to the Cali bot, add some code that meets your needs to the process_command.py.

All of the functions currently in use are available in that file.
